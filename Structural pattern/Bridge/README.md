<div align="center"><h1> Мост (Bridge) </h1></div>    

## Описание    
Декоратор - структурный шаблон проектирования, который позволяет отделить абстракцию от реализации.     
![Иллюстрация к проекту](https://github.com/FrostyCreator/Design-pattern/blob/master/Images/Bridge.gif)    

## Когда применять    
- Когда надо разделять абстракцию и реализацию так, чтобы они могли изменяться независимо.
